1768. Merge Strings Alternatively

1. Find the longer string among the two strings. 2. Convert the two strings to char array. 3. Create a stringbuilder so that chars of w1 and w2 can be appended. 4. Use a for loop to loop through w1 and w2 respectively and append them into the stringbuilder obj. 5. Append the extra characters (if any) to the end of the stringbuilder obj. 6. Convert the stringbuilder to string and return it.
